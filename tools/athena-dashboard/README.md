# ØK ATHENA Dashboard

Real-time penetration testing command center. Single-page web application powered by FastAPI + vanilla JS.

## Quick Start

```bash
./start.sh
# Dashboard: http://localhost:8080
```

## Architecture

| Component | File | Size |
|-----------|------|------|
| Backend API | `server.py` | ~14K lines, 153 endpoints |
| Frontend UI | `index.html` | ~17K lines, Chart.js + WebSocket |
| Agent Manager | `agent_session_manager.py` | Multi-agent orchestration |
| Agent Configs | `agent_configs.py` | 9 agent roles + prompts |
| Finding Pipeline | `finding_pipeline.py` | Dedup, fingerprinting, persistence |
| Message Bus | `message_bus.py` | Agent-to-agent pub/sub |
| Kali Client | `kali_client.py` | Dual backend HTTP client |

## Dashboard Features

- **KPI Cards** — Active engagements, hosts discovered, services, findings, exploit rate, TTFS, MTTE
- **Host Selector** — Per-engagement host filtering (sidebar toggle, scopes all widgets)
- **Agent Status Grid** — Real-time LED indicators for 9 AI agents
- **Findings by Severity** — Bar chart with critical/high/medium/low/info breakdown
- **Scan Coverage** — Ring chart showing PTES phase coverage percentage
- **Exploit Gauge** — Confirmed exploit rate with confirmed + unverified tracking
- **Kill Chain Depth** — Visual MITRE kill chain progression
- **Attack Chains** — Detected attack paths with linked findings
- **Remediation Priority** — Bubble chart (CVSS x Hosts) grouped by CVE
- **Credential Tracker** — Harvested credentials, default/weak, unique accounts
- **Attack Surface Radar** — Category distribution radar chart
- **PTES Methodology Matrix** — Phase-by-phase coverage with agent mapping
- **Attack Graph** — Interactive Neo4j-powered node graph
- **Evidence Gallery** — Screenshots and artifacts with filtering
- **AI Timeline** — Scrolling feed of agent actions and tool calls
- **Reports** — Auto-generated technical, executive, and remediation reports

## Dependencies

- Python 3.13+ with FastAPI/Uvicorn
- Neo4j 5.x (graph database)
- Kali Linux backend(s) with MCP server
- Anthropic API (Claude agents)

## Environment

Configured via `.env` (auto-generated by `start.sh` on first run) or 1Password CLI.
